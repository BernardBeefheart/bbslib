Je teste mon code avec plusieurs implémentations de Scheme. Voici quelques remarques à propos de chacune d'entre elles.

note préliminaire

Je travaille sous Debian. Certaines des notes qui suivent devront certainement être adaptées en particulier pour Windows, Mac OS X ou BSD.

les implémentations R7RS

Chibi Scheme

On le trouve sur Github. Les instructions de la page d'accueil sont suffisantes pour le compiler et l'installer. Son utilisation est très simple en ligne de commande. Pour une utilisation interactive: 

chibi-scheme -R -I . -I path/to/code

Pour lancer un programme:

chibi-scheme -I . -I path/to/code lecode.scm

L'option -I directory indique à Chibi Scheme où se trouve les bibliothèques à importer.

Note importante: les bibliothèques doivent avoir l'extension .sld. Une solution simple est de créer un lien dynamique sur le fichier .scm correspondant:

ln -s mylib.scm mylib.sld
