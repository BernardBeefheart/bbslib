Here is an experimental Scheme lib. Some code is taken from some of my (very) old C applications. I use R7RS Scheme systems unless for specific tools.

sl-printf

This printf was done to get formatted output as in C. Written for R7RS.

make-chicken-units

This utility is written for Chicken Scheme. It transform a standard scheme source file in a Chicken Scheme unit by adding a line at the beginning of the file :

(declare (unit unit-name))

tester

A simple unit testing system.
